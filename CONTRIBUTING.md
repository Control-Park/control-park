# Contributing 

Pre-requisites:
1. Install [node.js version over 20.19+ ](https://nodejs.org/en/download)

# To get started on this project, fork the repository:

1. git clone https://github.com/Control-Park/control-park

Next, navigate to the location of this project directory:

2. cd Control-Park

Install dependencies:

3. npm install
4. npm update

Start development server:

5. npm start
